Urban footprinter
===============================

Overview
--------

A convolution-based approach to detect urban extents from raster datasets.

Installation / Usage
--------------------

To install use pip:

    $ pip install urban-footprinter


Or clone the repo:

    $ git clone https://github.com/martibosch/urban-footprinter.git
    $ python setup.py install
